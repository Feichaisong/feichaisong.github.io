所见所感

发表于 2017-03-06 | | 老娘接待了 人
字数统计 | 阅读时长

2017-0306
回住处经过的偏弄，一个打扮时尚的奶奶勾搭着佝偻着背的爷爷，谈论着旅行中的所见所闻。我只觉得执子之手与子偕老是件让人羡慕的事。
即使他们曾经或许也有过各种磕磕碰碰，但在古稀之年依旧能相互搀扶着信步于月光下，人生或许真的没有更美好的事了吧。

2017-0404
17年又过了四分一，清明小长休又来到了终点，我又宅了三天。
认真想想，今年以来似乎都是一个人带着，偶尔出去转转也宁愿一个人，感觉过上了老年人的生活。我想自己会悲哀，但是却没有。
活着却如同死去的感觉应该就是如此了吧，明明能感知周围的一切，但你只能一个人静静感受，没有述说的对象，没有人会记得你。
可，我却对这种”恐惧“毫无畏惧，心死还是习惯孤独？

2017-0405
单纯为写点东西，既然说“写作”便是自己和内心的对话，那也至少说明有人说话。是的，好久没和人说话的感觉，想找个人聊聊。可是打开微信通讯录，看来看去要么太熟没啥可聊，要么太生也没啥可聊，还是和自己聊聊最好了。
说说隔壁的女人吧，其实内心里是想说女生，可突然觉得她不配啊。女生在我这里应该是干净卫生，生活规律有条理，谈吐中总能透露出内心的波澜的一类女性群体。
可住在隔壁的这个女人，昼伏夜出，让我一度怀疑是做色情服务的鸡，想想这种对他人的揣测是邪恶的，就放弃了。但在仅有的公共空间里，见其所作所为，实在是难以认同这种人。
作为男生，自觉环境卫生已经搞得很差了，但相比而言，从来没发现自己这么干净清爽过。我很是好奇，她的个人空间是怎样的？通过其装扮却是判若两人，实难相信这么一位还算靓丽的“女子”，竟然是如此邋遢之人。
“人不可貌相”果不其然呐！

2017-0409 雨风
住在群租房里，公共区域人多杂乱是不可免的。但凡住在里边自然也就想当然的当做是自己的”家”，所以也就毫无形象可言，在家里形象是可以无所顾忌的。只是突然觉得好笑，你觉得不可就近的女神可能只是一个穿着睡衣披头散发，在陌生人前尽情展示真实的“自我”的“糟糕女生”。
最近又搬进来了新的租户，小小的空间越加饱满了，而我又得开始习惯新的生活作息，同时也提醒了我一件事，又一年过去了。

2017-0410 阴雨
傍晚收到景儿要来的消息，我欣喜若狂。
一天的糟糕心情瞬间散去，昨晚因新来的邻居话多，被叨扰的无法入睡，一晚上没睡好今早醒来就觉得不爽。依旧是这么容易被周围的事物影响，改或许真的没那么容易。但今天这都不重要了，反正今天就开心，原来被需要的感觉这么爽。
这让我突然回到十年前，原来我和我的这些家人已经走过十年了。而所有的羁绊只源于当时几个人的无心组成的家庭，一路心心相惜即使常常相距甚远，可总是有一股隐藏的力量将我们联系着。无论何时，再相见，时光又回来了。因着相距的稀少和短暂，我格外的珍惜每一个好友的到访。
之前听说一旦听到好友到访有开心到的跳起来的心情，这这感觉我想就是当下的心情了吧！无形之中都感受到了自身的笑脸是多么的灿烂。朋自远方来，何其乐乎！
