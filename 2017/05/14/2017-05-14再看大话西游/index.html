再看大话西游

发表于 2017-05-14 | 分类于 电影 | | 老娘接待了 人
字数统计 | 阅读时长

首先得承认，我想再看大话是因为脑海中闪现了紫霞的容颜，这个并不纯洁的初衷。—-开个玩笑
其实因为看到了一段某节目上，“紫霞”听到大话的对白时失了魂的视频，我忍不住去看了这个节目，然后顺便回顾了下西游二片。
第一次看大话的时候是在央视6套，但已忘记是什么时间了。只隐约记得唐僧及其烦人的念念叨叨，然后笑笑就过去了。很长的时间里我并没有觉得它是一部称得上经典的作品，所以也没有重看的欲望。后来日常生活中出现了像“曾经有一分真挚的爱情放在我面前，我不懂的珍惜，直到失去后才后悔莫及…；爱不需要理由么？爱需要理由么；我的意中人是个盖世雄，他身披金甲圣衣，踏七彩祥云来娶我…”一些相当经典的对话
再看西游时已经是工作两年后了，单身男子不能出去犯罪，没有女朋友可以玩，只能宅家捣腾电脑玩自己然后”陶冶”情操了。其实一直以来，我觉得像我这种从来没有自己主见的人，我觉得看电影看几遍都一样，所以很少重看电影(当然小电影除外)。只是这次再看西游时，我却发现自己是笑不出来的，不免有些奇怪，到底是什么改变了我。其中吸引我的是一些看似玩笑，实则也是一些我不知什么时候开始想不明白的事。
比方说菩提老祖和至尊宝讨论到底爱需不需要理由。生活中大多事情都有个因为所以，可唯独爱恨这东西看似需要理由，但又往往不曾理由。紫霞的爱的理由是能拔出紫青宝剑的人便是她的意中人，看似好笑的理由，可为何总觉得这么个理由是坚不可摧的。
