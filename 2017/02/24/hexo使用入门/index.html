概述

Hexo is a fast,simple and powerful blog framework.你可以用Markdown或者其他语言创建文章，
而且仅需几秒钟就可以用beautiful theme生成静态网页。嗯，官网上就是这么说的。

环境准备

说明：本文的操作流程及步骤是基于window7平台进行的。

git安装

安装git作为把生成的静态网页推送到GitHub之用
Windows 平台下直接下载msysgit然后各种下一步即可。由于众所周知的【墙】，下载的时候可能会有点慢，所以最好找把梯子速度会快N倍。

node.js安装

node.js 则是用于生成静态网页
nodejs到官网直接下载安装即可

GitHub注册

请先到GitHub中注册一个账号，只需用邮箱注册就OK了。
GitHub 则是用作于博客服务器

hexo安装

在任意目录下创建一个用于存放博客的文件夹如hexo，为了避免一些未知的错误，路径中最好不要出现中文。
创建好文件夹后，在新建的文件夹上右键选择git bash here，然后在跳出的命令框中输入以下命令安装hexo

安装好hexo后对博客进行初始化
